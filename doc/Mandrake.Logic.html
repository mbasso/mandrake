    both(first_func, second_func)

    Returns true if both functions return true

    Examples

    iex>  my_function = Mandrake.Logic.both(fn x -> x > 10 end, fn x -> Kernel.rem(x, 2) == 0 end)
...>  my_function.(100)
true
iex>  my_function.(101)
false
